# 🚗 Car Price Predictor

A  **Machine Learning web application** that predicts the resale price of a used car based on key features like brand, model, year of purchase, fuel type, and kilometers driven.

Built using **Flask**, **scikit-learn**, and deployed on **Render**.

---

## ✨ Features

- 🔍 Predicts market value of used cars
- 🏭 Dynamic company → model dropdown
- 📅 Year-wise price consideration
- ⛽ Fuel type based prediction
- 🧠 Machine Learning model (Linear Regression)
- 🎨 Clean, modern, responsive UI
- ☁️ Deployed on Render

## 📊 Machine Learning Model

- **Algorithm:** Linear Regression  
- **Input Features:**
  - Car Company
  - Car Model
  - Year of Purchase
  - Kilometers Driven
  - Fuel Type
- **Output:** Predicted resale price (Rs)

The model was trained on a cleaned used-car dataset and serialized using `pickle`.
